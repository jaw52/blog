# yarn

### 安装

```js
npm install vue
yarn add vue
```

安装全局包

```js
npm install -g vue
yarn global add vue
```

### 移除

```js
npm uninstall vue
yarn remove vue
```

移除全局包

```js
npm uninstall -g vue
yarn global remove vue
```

### 升级

```js
npm update vue
yarn upgrade vue
```

升级全局包

```js
npm update -g vue
yarn global upgrade vue
```
